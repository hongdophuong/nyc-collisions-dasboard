NYC Traffic Collisions Analysis Dashboard

1- Overview: 
This project analyzes traffic collision data in New York City from 2021 to 2023.
The goal is to identify trends, peak collision periods, and high-risk contributing factors using an interactive Excel dashboard.

2- Tools: Excel (Pivot Tables, Pivot Charts, Slicers, Conditional Formatting)

3- Dataset: Maven Analytics/NYC Motor Vehicle Collisions (Open Data)

4- Key Features: 
- Interactive dashboard with slicers (Year, Borough, Contributing Factor)
- KPI cards showing total collisions and dangerous collision rate
- Time-series analysis of monthly collision trends
- Heatmap visualization of collisions by hour and weekday
- Analysis of contributing factors and their associated risk levels

5- Key Insights: 
- Collision frequency peaks during evening rush hours (4–7 PM)
- Weekend nights show higher collision activity compared to weekdays
- "Failure to Yield Right-of-Way" has the highest dangerous collision rate (64%)
